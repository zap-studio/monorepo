## What is Zap.ts?

Zap.ts is a modern, full-stack TypeScript monorepo starter kit designed to help you build fast with ease. It comes with everything you need to launch your next project.

## Features

- **Authentication** - Complete auth system with multiple providers using Better Auth
- **Email** - Transactional emails with React Email templates and Resend
- **Payments** - Polar integration for subscriptions and billing
- **Database** - Type-safe database access with Drizzle ORM
- **UI Components** - Pre-built components with shadcn/ui
- **Analytics** - User tracking with PostHog
- **Internationalization** - Built-in i18n support
- **Deployment** - Deploy anywhere with Vercel or Docker
