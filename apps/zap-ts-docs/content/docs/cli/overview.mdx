# Overview

Zap.ts includes a CLI tool that helps you manage your project with simple, powerful commands. The CLI is available via the `@zap-ts/cli` npm package, which provides the alias `zap`.

## Installation

We recommend you to install the CLI globally for easier access.

<Tabs items={['npm', 'yarn', 'pnpm', 'bun']}>
<Tab value="npm">
```bash
npm install -g @zap-ts/cli
```
</Tab>
<Tab value="yarn">
```bash
yarn global add @zap-ts/cli
```
</Tab>
<Tab value="pnpm">
```bash
pnpm add -g @zap-ts/cli
```
</Tab>
<Tab value="bun">
```bash
bun add -g @zap-ts/cli
```
</Tab>
</Tabs>
